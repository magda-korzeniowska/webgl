# 3D Animation

**Demo:** https://webgl-daft.herokuapp.com/

### Installation

1. Clone repository using:
    `$ git clone https://github.com/magda-korzeniowska/webgl.git` 

2.  Install all dependencies:
    `$ npm install` or `$ yarn install`

3. Run application:
    `$ npm start` or `$ yarn start`
    
### Technologies:
* JavaScript 
* three.js library
* Express
